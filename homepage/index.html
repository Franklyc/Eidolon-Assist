Requirements

  • Python 3.7+
  • PyQt6
  • Groq API key (for Whisper transcription)
  • OpenAI-compatible API key (for AI responses)

Installation

# Clone the repository git clone https://github.com/franklyc/eidolon-assist.git

# Install dependencies pip install -r requirements.txt

# Run the application python eidolon_assist.py

About Eidolon Assist

Eidolon Assist is a personal assistant application designed to provide a modern, seamless interface for interacting with advanced AI models. Combining voice, text, and image capabilities in a elegant UI with rounded corners, gradients, and blur effects.

Built with Python and PyQt6, this open-source project aims to make AI interaction more accessible and natural. The application supports various language models and can be customized to fit your specific needs.
